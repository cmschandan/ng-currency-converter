# ng-currency-converter

## Create a New Project
  `ng new currency_converter`
  
## Serve application 
  `ng serve`
  
 ## create a service 
  `ng g s service_name`
  
## Api URL 
  `https://api.exchangerate.host/latest?base=USD`
